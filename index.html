🧪 Lab 4 – Express Server Basics

This lab introduced the basics of setting up a Node.js server using the Express framework. You learned how to configure middleware, define routes, parse JSON requests, and respond to API calls using fetch().

🔍 Topics Covered

  • Creating a basic Express server
  • Using express.json() middleware
  • Defining POST routes to handle client input
  • Testing with a simple HTML form and fetch()

📁 Key Files

  • lab-04.js – Express server logic
  • package.json – Dependencies
  • index.html – Frontend form and client-side logic
